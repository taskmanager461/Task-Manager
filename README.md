# 🎯 Self-Trust Score - Task Manager

Το **Self-Trust Score** είναι μια εφαρμογή διαχείρισης εργασιών (Task Manager) που επικεντρώνεται στη συνέπεια. Είναι ένας "κινητήρας συνέπειας" για τις υποσχέσεις που δίνετε στον εαυτό σας, υπολογίζοντας ένα σκορ εμπιστοσύνης με βάση την ολοκλήρωση των εργασιών σας.

## 🚀 Χαρακτηριστικά
- **Σύστημα Σκορ Εμπιστοσύνης**: Υπολογισμός ημερήσιου σκορ βάσει ολοκληρωμένων εργασιών και δυσκολίας.
- **Πολυγλωσσικό UI**: Πλήρης υποστήριξη για Ελληνικά, Αγγλικά, Ισπανικά, Γαλλικά, Γερμανικά, Ιταλικά και Πορτογαλικά.
- **Dark/Light Mode**: Πλήρης υποστήριξη θέματος για άνετη χρήση.
- **Αναλυτικά Διαγράμματα**: Οπτικοποίηση της προόδου σας με Plotly.
- **PWA Support**: Δυνατότητα εγκατάστασης της εφαρμογής στην αρχική οθόνη του κινητού.
- **Ειδοποιήσεις**: Υπενθυμίσεις για εκκρεμείς εργασίες και ημερήσιες αναφορές.

## 🛠️ Τρόπος Λειτουργίας (Local)

### 1. Προετοιμασία
Βεβαιωθείτε ότι έχετε εγκαταστήσει την Python 3.10+.

### 2. Εγκατάσταση Εξαρτήσεων
```bash
pip install -r requirements.txt
```

### 3. Ρύθμιση Περιβάλλοντος
Δημιουργήστε ένα αρχείο `.env` στη ρίζα του project (χρησιμοποιήστε το `.env.example` ως πρότυπο).

### 4. Εκτέλεση Backend (FastAPI)
```bash
uvicorn backend.main:app --reload
```

### 5. Εκτέλεση Frontend (Streamlit)
```bash
streamlit run frontend/app.py
```

---

## 🌐 Deployment (Ανάπτυξη)

### Επιλογή Α: Render (Προτεινόμενο)
1. Συνδέστε το GitHub repository σας στο **Render**.
2. Χρησιμοποιήστε το αρχείο `render.yaml` για αυτόματη δημιουργία των υπηρεσιών (Blueprint).
3. Προσθέστε τις απαραίτητες μεταβλητές περιβάλλοντος (Environment Variables) στο dashboard του Render.

### Επιλογή Β: Docker
Η εφαρμογή περιλαμβάνει `Dockerfile` και `docker-compose.yml`.
```bash
docker compose up --build
```

## 📂 Δομή Project
- `backend/`: API (FastAPI), Μοντέλα βάσης δεδομένων, JWT Auth.
- `frontend/`: UI (Streamlit), Μεταφράσεις, Στυλ.
- `config/`: Ρυθμίσεις εφαρμογής.
- `static/`: Εικονίδια και αρχεία PWA.

## 🛡️ Ασφάλεια
- Τα συνθηματικά κρυπτογραφούνται με **bcrypt**.
- Η ταυτοποίηση γίνεται μέσω **JWT tokens**.
- Υποστήριξη HTTPS για παραγωγικό περιβάλλον.
